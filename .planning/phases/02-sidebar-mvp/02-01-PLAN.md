Key constraints:
- storage.sync: 100KB total, 8KB per item — user settings and group metadata only
- storage.local: 10MB default — tab snapshots, saved entries, session state
- CONSTANTS.STORAGE_KEYS: { SESSION_STATE, SETTINGS, WORKSPACES, USER_RULES, TIMESTAMPS }
- CONSTANTS.DEFAULT_SETTINGS: { t1Minutes:5, t2Minutes:15, t3Days:7, autoGroup:true, persistSessions:true, managePinned:false, hoverPreRender:true, showRamSavings:true, batchSize:3, whitelist:[] }
- SESS-01: auto-save debounced at 30 seconds — scheduleSave() must use a timer-based approach
- SESS-02: on startup, background reads saved state and restores sidebar — loadState() must return the last saved session blob intact
- Pattern: IIFE + globalThis export (same as browser-adapter.js, constants.js, lifecycle-manager.js)
- Pattern: module.exports for Node.js test compat (same dual-export pattern across all core/ modules)
- StorageManager must NEVER throw — always try/catch internally, log errors, return null/empty on failure

Session state schema (the blob saved to storage.local):
```javascript
{
  savedEntries: [],   // SavedTabEntry[] — Stage 3/4 entries (not in tabRegistry)
  groups: [],         // TabGroup[] — group metadata (name, color, order, isCollapsed)
  timestamp: 0        // Date.now() when last saved
}
```

TabGroup shape (from CONSTANTS.DEFAULT_GROUPS pattern):
```javascript
{ id, name, color, order, isCollapsed, isCustom, domainRules: [] }
```

SavedTabEntry shape (Stage 3/4 entries only):
```javascript
{ savedId, url, title, favicon, groupId, stage, savedAt, navigationHistory: [], isStateful }
```

BrowserAdapter interface (from core/browser-adapter.js):
```javascript
BrowserAdapter.storage.local.get(keys)    // → Promise<object>
BrowserAdapter.storage.local.set(items)   // → Promise<void>
BrowserAdapter.storage.sync.get(keys)     // → Promise<object>
BrowserAdapter.storage.sync.set(items)    // → Promise<void>
```

       * Debounced save — call on every lifecycle/group change event.
       * Resets the 30s timer on each call; writes only once when timer fires.
       * SESS-01: auto-saves every 30 seconds maximum.

      function scheduleSave(state) { ... }

      /**

      async function getSettings() { ... }
