Key constraints:
- Extension CSP: no inline scripts, no inline event handlers (onclick="...", no script src from external origins)
- Sidebar width: 300px minimum, 500px maximum — must work at all widths in this range
- Sidebar height: fills the browser sidebar panel (100vh); main group list scrolls independently
- Group colors: 10 categories from CONSTANTS.DEFAULT_GROUPS — #1B6B93 (dev), #E67E22 (work), #E91E8C (social), #27AE60 (shopping), #8E44AD (entertainment), #E74C3C (news), #00ACC1 (research), #F39C12 (finance), #009688 (lifestyle), #95A5A6 (other)
- Stage indicators: green dot (active), blue dot (discarded), bookmark icon (saved), clock icon (archived)
- Performance: UI-11 requires < 200ms initial render — avoid CSS that forces layout recalc on every item (no box-shadow on every tab-entry, use will-change sparingly)
- Font stack: system-ui, -apple-system, sans-serif (extension must not load external fonts)
- Vanilla JS: no framework — all class names must be manipulatable with classList.add/remove

DOM contract for sidebar.js (these IDs and class names are the API):
- #tn-header — header container
- #tn-search — search input
- #tn-ram-indicator — RAM savings display span
- #tn-settings-btn — settings gear button
- #tn-group-list — main group card mount point
- #tn-archive — archive section mount point
- #tn-tab-count — footer count span
- #tn-save-all-btn — "Save & Close All Inactive" button

Group card HTML template (replaces existing static content; sidebar.js builds these dynamically):
```html
<div class="tn-group-card" data-group-id="dev" data-collapsed="false">
  <div class="tn-group-header">
    <span class="tn-group-color-bar" style="--group-color: #1B6B93"></span>
    <span class="tn-group-name">Dev</span>
    <span class="tn-group-count">3 open · 8 saved</span>
    <button class="tn-collapse-btn" aria-label="Collapse Dev group" aria-expanded="true">&#9660;</button>
  </div>
  <ul class="tn-tab-list" role="list">
    <!-- tab entries appended here -->
  </ul>
</div>
```

Tab entry HTML template:
```html
<li class="tn-tab-entry" data-tab-id="123" data-stage="active" role="listitem">
  <img class="tn-favicon" src="" alt="" width="16" height="16" />
  <div class="tn-tab-info">
    <span class="tn-tab-title" title="Full Title Here">Truncated Title</span>
    <span class="tn-tab-domain">github.com</span>
  </div>
  <span class="tn-stage-indicator" aria-label="Active"></span>
  <div class="tn-tab-actions">
    <button class="tn-action-btn tn-action-discard" aria-label="Discard tab" title="Discard">&#128473;</button>
    <button class="tn-action-btn tn-action-close" aria-label="Save and close tab" title="Save &amp; Close">&#10006;</button>
  </div>
</li>
```

For saved/archived entries (data-stage="saved" or data-stage="archived"):
- Replace tn-action-discard with nothing
- tn-action-close becomes tn-action-restore with "Restore" label
- Clicking the entry (or restore button) triggers RESTORE_TAB

CSS requirements:
- .tn-group-card[data-collapsed="true"] .tn-tab-list { display: none } — collapse via data attribute
- .tn-tab-entry[data-stage="active"] .tn-stage-indicator { background-color: #27AE60; border-radius:50%; width:8px; height:8px }
- .tn-tab-entry[data-stage="discarded"] .tn-stage-indicator { background-color: #2196F3 }
- .tn-tab-entry[data-stage="saved"] .tn-stage-indicator — use a bookmark unicode character (content: "🔖" via ::before or SVG inline)
- .tn-tab-entry[data-stage="archived"] .tn-stage-indicator — use a clock character (content: "🕐" or similar)
- .tn-tab-title — text-overflow: ellipsis; white-space: nowrap; overflow: hidden
- Tab action buttons are hidden by default, shown on .tn-tab-entry:hover
- Header is fixed at top, footer is fixed at bottom, #tn-group-list scrolls in between

  <action>
    Replace the existing sidebar/sidebar.css stub with a complete CSS file. The CSS must cover: sidebar layout (fixed header/footer, scrolling main), group card structure, tab entry layout, stage indicator styles, collapse behavior, and responsive width behavior.

    CSS architecture requirements:
    1. **Root variables** — define the 10 group colors as CSS custom properties, plus spacing/font tokens
    2. **Layout** — body is a flex column; header and footer are fixed height; #tn-group-list fills remaining space with overflow-y: auto
    3. **Group card** — white background, left colored border using --group-color variable, margin-bottom between cards
    4. **Group header** — flex row: color bar (4px wide, full card height), group name (flex-grow), count badge, collapse button
    5. **Tab entry** — flex row: favicon (16px), info block (flex-grow, title + domain), stage indicator, action buttons
    6. **Title truncation** — text-overflow: ellipsis; white-space: nowrap; overflow: hidden on .tn-tab-title
    7. **Stage indicators** — data-stage attribute selectors for color and icon
    8. **Action buttons** — hidden by default (opacity: 0), shown on .tn-tab-entry:hover (opacity: 1)
    9. **Collapse** — .tn-group-card[data-collapsed="true"] .tn-tab-list { display: none }
    10. **Responsive** — sidebar works at 300px width minimum; group card title truncates before wrapping
    11. **RAM badge** — pill-shaped badge in header with green background
    12. **Footer** — flex row with count on left, button on right; button styled as primary action
    13. **Archive section** — similar card style but with toggle button

    Write the CSS file. Key selectors required:

    ```css
    /* Root & Layout */
    :root { --sidebar-min-width: 300px; --sidebar-max-width: 500px; ... }
    * { box-sizing: border-box; }
    body { min-width: var(--sidebar-min-width); max-width: var(--sidebar-max-width); display: flex; flex-direction: column; height: 100vh; margin: 0; }
    #tn-header { flex-shrink: 0; ... }
    #tn-group-list { flex: 1; overflow-y: auto; padding: 8px; }
    #tn-footer { flex-shrink: 0; ... }

    /* Group card */
    .tn-group-card { ... border-left: 4px solid var(--group-color, #95A5A6); ... }
    .tn-group-card[data-collapsed="true"] .tn-tab-list { display: none; }
    .tn-group-header { display: flex; align-items: center; ... }
    .tn-group-name { flex: 1; font-weight: 600; ... }

    /* Tab entry */
    .tn-tab-entry { display: flex; align-items: center; gap: 6px; padding: 5px 8px; ... }
    .tn-tab-info { flex: 1; min-width: 0; } /* min-width:0 enables ellipsis in flex children */
    .tn-tab-title { display: block; white-space: nowrap; overflow: hidden; text-overflow: ellipsis; }
    .tn-tab-domain { display: block; font-size: 11px; color: #888; ... }

    /* Stage indicators */
    .tn-stage-indicator { width: 8px; height: 8px; border-radius: 50%; flex-shrink: 0; }
    .tn-tab-entry[data-stage="active"] .tn-stage-indicator { background-color: #27AE60; }
    .tn-tab-entry[data-stage="discarded"] .tn-stage-indicator { background-color: #2196F3; border-radius: 2px; }
    .tn-tab-entry[data-stage="saved"] .tn-stage-indicator { background: none; font-size: 12px; width: auto; }
    .tn-tab-entry[data-stage="saved"] .tn-stage-indicator::before { content: "\1F516"; } /* bookmark */
    .tn-tab-entry[data-stage="archived"] .tn-stage-indicator::before { content: "\1F554"; } /* clock */

    /* Action buttons — hidden until hover */
    .tn-tab-actions { display: flex; gap: 2px; opacity: 0; transition: opacity 0.15s; }
    .tn-tab-entry:hover .tn-tab-actions { opacity: 1; }
    .tn-tab-entry:focus-within .tn-tab-actions { opacity: 1; }
    ```
